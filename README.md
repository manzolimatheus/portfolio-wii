## Como executar?

Primeiramente, instale os módulos necessários

```bash
npm i
```

Após, execute o projeto:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```

Abra [http://localhost:3000](http://localhost:3000) em seu navegador para ver a aplicação iniciada.

# Pontos importantes
- O Projeto é focado em front-end;
